# Chapter 1 Exercise Pack — Introduction to Claude Cowork Automation

**Course:** cowork-automation-tracy-2026
**Chapter:** 1 — Introduction to Claude Cowork Automation
**Total time budget:** 35 minutes
**Track:** Novice
**Scenario thread:** scenario-01 — Tracy's weekly project status report from Jira, Confluence, Slack, and Google Meet transcripts

## Learning Outcomes Addressed

- **LO-01.1:** Identify what Claude Cowork can and cannot automate in a program management context.
- **LO-01.2:** Distinguish skills, plugins, and agents at a conceptual level.
- **LO-01.3:** Recognize the 5-part RSTRM framework by applying it criterion-by-criterion to a real workflow.

## Prerequisites

- Read Chapter 1 through Section 7 (Worked Example).
- No software installation is required for Exercises 01 or 02.
- Exercise 03 requires no external tools; all work is written analysis in a Markdown file.

## Pack Structure

| Exercise | Stage | Scenario | Focus | Time |
|---|---|---|---|---|
| `worked-example/` | Fully solved walkthrough | scenario-01: weekly status report | Applying RSTRM to Tracy's highest-value workflow | 12 min |
| `exercise-02/` | Completion (partial scaffold) | scenario-04: email inbox triage | Filling in an RSTRM analysis and checkpoint specification | 15 min |
| `exercise-03/` | Independent (diagnosis) | scenario-01: status report skill | Diagnosing a silently failing skill configuration | 8 min |

Complete exercises in order. Each exercise builds on the prior one. Do not skip to Exercise 03 before attempting Exercise 02.

## Completion Guidance

- For the worked example, read every section including the Decision callouts. Do not skim.
- For Exercise 02, open the starter file before opening this README a second time. Attempt every TODO before consulting the solution.
- For Exercise 03, write your diagnosis before checking the verify script. The verify script confirms structure, not correctness.
- The debrief (`debrief.md`) contains the full reflection question set. Plan 5 minutes for it after completing all three exercises.
